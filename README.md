Para levantar el proyecto basta con ejecutar el script ubicado en `database/create.sql` dentro de `phpmyadmin`.

Luego instalamos las dependencias y ejecutamos el servidor en modo desarrollador:

```powershell
npm install
npm run dev
```

Y el proyecto se desplegara en http://localhost:3000.

> [!Note]
> Para editar un elemento basta con hacerle click a este elemento en la tabla en /lista.
